

MP3 Libraries

- Javazoom JLayer (LGPL) http://www.javazoom.net/javalayer/javalayer.html

VorbisSPI (LGPL) http://www.javazoom.net/vorbisspi/vorbisspi.html

MP3SPI and VorbisSPI automatically registers decoder/converter to Java Sound API when in class path.
